# Vercel Design System Skill

This skill configures the Antigravity agent to follow Vercel's stark, high-fidelity, black-and-ink styling guidelines.

## Layout Principles
1. **Canvas:** White or near-white canvas (#ffffff, #fafafa).
2. **Text:** High-contrast ink (#171717) for headers and primary body text, dark grey (#4d4d4d) for paragraphs, and mute gray (#888888) for labels.
3. **Typography:** Use Geist or Inter for displays/body, and Geist Mono for caption-mono and code blocks. Use strict letter-spacing modifications.
4. **Borders:** Thin, hairline borders (#ebebeb) instead of shadows for layout cards, or light dividers.
5. **Accents:** Restrained decorative colors. Use multi-color mesh gradient overlays (cyan, blue, magenta, amber) at hero scale, or subtle hover state transitions.

## Elements
- **Buttons:** Round pills or small cards with tight padding and black/white hover scales.
- **Interactive Form Fields:** High-fidelity outline inputs with small rounded corners (#ebebeb border) and active focus rings.
